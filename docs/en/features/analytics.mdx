---
title: "Analytics"
description: "Chat statistics and activity insights"
---

BotMux provides per-chat analytics accessible from the chat detail view.

## Metrics

- **Total message count** — lifetime messages in the chat
- **Daily activity** — messages today
- **Weekly active users** — unique senders in the past 7 days

## Charts

- **Hourly activity distribution** — 24-hour chart showing message frequency over a 7-day window

## Leaderboard

- **Top 10 contributors** — most active users by message count
