<Update label="2026-05-24" description="Week of May 24" tags={["Fixes"]}>

## Bug fixes

- **Most recent messages now appear at the top** — Message lists, text search results, and chat previews are now sorted by send time instead of by message ID. Freshly received messages no longer get hidden behind older messages from chats with higher ID counters, so [message monitoring](/docs/en/features/message-monitoring) always reflects what arrived most recently.

</Update>
