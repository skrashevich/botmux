---
title: "Переменные окружения"
description: "Настройка BotMux через переменные окружения"
---

| Переменная | Эквивалентный флаг | Описание |
|------------|-------------------|----------|
| `TELEGRAM_BOT_TOKEN` | `-token` | Токен Telegram-бота |
| `TELEGRAM_API_URL` | `-tg-api` | Пользовательский базовый URL Telegram API |
| `DEMO_MODE` | `-demo` | Установите `true` для демо-режима |
| `BOTMUX_ALLOW_AUTO_REGISTER` | — | Установите `1` для разрешения автоматической регистрации неизвестных токенов через перехват `setWebhook` в прокси API. По умолчанию отключено. |
| `BOTMUX_STRICT_WEBHOOK` | — | Установите `1` для обязательного использования HTTPS для всех URL вебхуков, включая адреса обратной петли. Если не установлено, HTTP разрешён для `localhost`, `127.0.0.1` и `::1`. |

## Использование

```bash
export TELEGRAM_BOT_TOKEN="123456:ABC-DEF..."
export TELEGRAM_API_URL="http://localhost:8081"
./botmux
```

Переменные окружения и флаги командной строки можно комбинировать. Флаги командной строки имеют приоритет.
