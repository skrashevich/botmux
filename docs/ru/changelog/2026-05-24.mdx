## Исправления

- **Самые свежие сообщения теперь показываются сверху** — Списки сообщений, результаты текстового поиска и превью чатов теперь сортируются по времени отправки, а не по идентификатору сообщения. Только что полученные сообщения больше не прячутся за более старыми сообщениями из чатов с большими значениями счётчика ID, поэтому [мониторинг сообщений](/docs/ru/features/message-monitoring) всегда отражает то, что пришло самым последним.

</Update>
