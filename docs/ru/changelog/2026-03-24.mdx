## Новые функции

- **Профили пользователей** — Нажмите на любое имя пользователя в панели управления, чтобы просмотреть модальное окно профиля со статистикой и последними действиями. Имена пользователей теперь кликабельны в списке сообщений и пользователей.

- **Уведомления об обновлениях** — BotMux теперь проверяет GitHub на наличие новых релизов и отображает закрываемый баннер при наличии обновления. Вы также можете вручную проверить обновления через футер панели управления. Подробнее о [развертывании](/docs/ru/deployment/requirements).

- **Перехват вебхуков** — BotMux теперь перехватывает вызовы API `setWebhook`, `deleteWebhook` и `getWebhookInfo`, проходящие через [API-прокси](/docs/ru/features/api-proxy). Это означает, что боты автоматически настраиваются, когда ваш бэкенд устанавливает вебхук, без ручной настройки.

- **Новый брендинг** — У BotMux новый логотип Hub, отражающий функцию мультиплексора. Логотип отображается в боковой панели, на экране входа и в фавиконе браузера.

- **Вынесенные переводы** — Переводы интерфейса теперь загружаются из внешних файлов, а не встроены в код. Это упрощает добавление новых переводов и делает код интерфейса чище. Английский и русский языки доступны из коробки. Подробнее в разделе [i18n](/docs/ru/features/i18n).

## Улучшения

- **Версия в футере** — Футер панели управления теперь отображает текущую версию BotMux с кнопкой для ручной проверки обновлений.

- **Улучшенный просмотрщик логов** — [Просмотрщик логов приложения](/docs/ru/features/analytics) стал больше, отображает имена ботов рядом с записями и использует улучшенную цветовую кодировку для удобного чтения.

- **Улучшенный длинный опрос** — Улучшен разбор параметров и обработка обновлений для [длинного опроса](/docs/ru/features/long-polling). Интерфейс теперь показывает более понятные подсказки по настройке длинного опроса, а пересылка на бэкенд автоматически пропускается, когда длинный опрос включен для бота.

- **Улучшенная загрузка медиа** — [Прокси для файлов](/docs/ru/features/media-support) Telegram теперь обрабатывает загрузку медиафайлов более надежно, с улучшенным извлечением файлов и доставкой контента.

## Исправления ошибок

- **Исправлена блокировка базы данных под нагрузкой** — Добавлен тайм-аут ожидания для подключения к SQLite, предотвращающий ошибки блокировки при одновременных запросах к базе данных.

- **Исправлен дублирующийся опрос** — BotMux теперь блокирует одновременные вызовы `getUpdates` для одного и того же бота, предотвращая конфликты, когда прокси и клиент длинного опроса одновременно пытаются получить обновления.

- **Исправлен разбор параметров длинного опроса** — Переход на структурированный разбор JSON для параметров запросов длинного опроса, устраняющий крайние случаи, когда смещения или лимиты обновлений применялись некорректно.
